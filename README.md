- 제외하고자 하는 태그는 아래와 같이 작성한다.
```
language:italian
language:czech
language:dutch
language:german
```

- 북마크는 hentoid의 메타데이터 내보내기의 데이터를 사용한다.

- 파이썬 라이브러리는 아래와 같다.    
<code>pip install -r requirements.txt</code> 로 한번에 설치할 수 있다. 
```
certifi==2023.7.22
charset-normalizer==3.3.2
idna==3.4
parmap==1.7.0
PyQt5==5.15.10
PyQt5-Qt5==5.15.2
PyQt5-sip==12.13.0
requests==2.31.0
urllib3==2.0.7
```

- 이 프로그램에는 다운로드 기능은 없다. hentoid에 다시 옮겨서 북마크를 복구하도록 하자.
